Rhonda Goolsby

Frontend Web Developer

My path to frontend web development wasn’t traditional, but it was driven by curiosity and a passion for creativity. I was searching for a challenge that would let me build visually engaging, interactive experiences—and I found that in coding. Since then, I’ve discovered the satisfaction of writing code that brings design to life.

Over the past few years, I’ve developed responsive, accessible web applications using HTML, CSS, and JavaScript/Jquery often collaborating closely with designers, digital marketing and backend teams. I enjoy turning complex ideas into clean, intuitive user interfaces and continuously exploring new tools to stay sharp and deliver polished digital experiences.
